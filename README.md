<p dir="auto">We recommend using Dawn as a starting point for theme development. <a href="https://shopify.dev/themes/getting-started/create" rel="nofollow">Learn more on Shopify.dev</a>.</p>
<blockquote>
<p dir="auto">If you're building a theme for the Shopify Theme Store, then you can use Dawn as a starting point. However, the theme that you submit needs to be <a href="https://shopify.dev/themes/store/requirements#uniqueness" rel="nofollow">substantively different from Dawn</a> so that it provides added value for merchants. Learn about the <a href="https://shopify.dev/themes/tools/dawn#ways-to-use-dawn" rel="nofollow">ways that you can use Dawn</a>.</p>
</blockquote>
<p dir="auto">Please note that the main branch may include code for features not yet released. The "stable" version of Dawn is available in the theme store.</p>
